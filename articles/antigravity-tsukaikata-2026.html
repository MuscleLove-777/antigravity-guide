本題

Antigravityとは？Googleが描く開発の未来

Antigravityは、Googleが開発する「AIファースト」な統合開発環境（IDE）です。従来のIDEがコードエディタを中心に据えていたのに対し、AntigravityはAIエージェントを開発の中心に置き、人間とAIが協調して開発を行う新しいパラダイムを提唱しています。2026年現在、その特徴は以下の点に集約されます。

  • AIエージェントによる自律コーディング: 単なるコード補完や生成に留まらず、開発タスク全体（機能追加、バグ修正、テスト実装など）を理解し、計画を立て、自律的にコードを書き、テストし、レビューを提案します。
  • マルチLLM対応: Google独自のGeminiシリーズだけでなく、Anthropic Claude、OpenAI GPTなど、複数の大規模言語モデル（LLM）をプラグイン形式でサポート。タスクやコストに応じて最適なLLMを使い分けることが可能です。
  • クラウドネイティブ設計: 全ての開発環境がクラウド上で動作するため、どこからでもアクセス可能。強力な計算リソースを背景に、複雑なAI処理や大規模なビルドも高速に実行します。
  • VS Code拡張とWeb版の選択肢: 慣れ親しんだVS Codeの拡張機能として、またはWebブラウザから直接アクセスできるクラウドIDEとして、開発スタイルに合わせて柔軟に利用できます。

Antigravityは、単なるツールの進化ではなく、「開発のあり方」そのものを再定義しようとしています。AIが開発の共同パートナーとなる時代において、その中心的役割を担うのがAntigravityなのです。

Antigravityの始め方：セットアップから基本操作まで

Antigravityを使い始めるのは非常に簡単です。以下の手順でセットアップと基本的な操作をマスターしましょう。

1. Antigravityのインストールとアクセス

  • VS Code拡張版の場合:
    1. VS Codeを開き、拡張機能マーケットプレイスで「Antigravity」を検索します。
    2. 「Google Antigravity IDE」拡張機能をインストールします。
    3. インストール後、VS CodeのサイドバーにAntigravityのアイコンが表示されます。
    4. アイコンをクリックし、Googleアカウントでログインして認証を完了させます。
    5. 利用規約に同意し、初期設定ウィザード（言語モデルの選択、AIエージェントの基本設定など）を進めます。
  • クラウドIDE（Web版）の場合:
    1. Webブラウザを開き、Antigravityの公式サイト（antigravity.google.com）にアクセスします。
    2. Googleアカウントでログインします。
    3. VS Code拡張版と同様に、初期設定ウィザードを進めます。

2. プロジェクトの作成・インポート

  • 新規プロジェクトの作成:
    1. Antigravityのメイン画面で「New Project」を選択します。
    2. プロジェクト名、使用する言語/フレームワーク（例: Python/FastAPI、TypeScript/React）、およびAIエージェントのタイプ（例: Web開発エージェント、MLエージェント）を指定します。
    3. 「Create Project」をクリックすると、指定されたテンプレートとエージェント設定で開発環境が構築されます。
  • 既存プロジェクトのインポート:
    1. 「Import Project」を選択します。
    2. Gitリポジトリからインポートする場合: リポジトリのURL（GitHub, GitLab, Bitbucketなど）を入力し、認証情報を設定します。Antigravityは自動的にリポジトリをクローンし、開発環境を構築します。
    3. ローカルフォルダからインポートする場合（VS Code拡張版のみ）: 既存のVS CodeワークスペースをそのままAntigravity環境として利用できます。クラウドストレージへの同期オプションも選択可能です。

3. UIの概要 AntigravityのUIは直感的で、以下の主要パネルで構成されます。

  • コードエディタ: VS Codeベースの強力なコードエディタ。シンタックスハイライト、コード補完、デバッグ機能など、おなじみの機能が利用できます。
  • Chatパネル: AIエージェントと対話するためのメインインターフェース。自然言語でタスクを指示したり、質問したりします。
  • Manager View: AIエージェントの思考プロセス、計画、実行ステップ、変更履歴などを可視化するパネル。後述の「エージェント機能活用テクニック」で詳しく解説します。
  • ファイルエクスプローラー: プロジェクトのファイル構造を管理します。
  • ターミナル: 統合されたターミナルでコマンドを実行します。

4. 最初のAIアシスト：コード生成とデバッグ プロジェクトを開いたら、早速AIエージェントを使ってみましょう。

  • コード生成の指示:
    1. Chatパネルを開きます。
    2. 「Web APIのエンドポイント/usersを作成し、ユーザーリストを返すPythonコードを生成してください。」のように、自然言語でタスクを指示します。
    3. エージェントはコードを生成し、エディタに提案します。コードの変更内容を確認し、「Apply Changes」で適用します。
  • デバッグの依頼:
    1. コードにエラーがある場合、Chatパネルで「このコードのバグを修正してください。」と指示します。
    2. エージェントはエラーログを解析し、問題を特定。修正案を提示し、必要であればテストコードも生成してバグが修正されたことを確認します。

Antigravityの基本操作：AIとの協調開発ワークフロー

Antigravityは、AIエージェントとの密な協調を通じて開発を進めることを前提としています。効果的なワークフローを見ていきましょう。

1. プロンプトの出し方と効果的な指示 AIエージェントを最大限に活用するには、明確で具体的なプロンプトが重要です。

  • タスクの明確化: 「〜機能を追加してください」「〜バグを修正してください」のように、何をしてほしいかを明確に伝えます。
  • コンテキストの提供: 関連するファイルやコードスニペットを参照させたり、「既存のアーキテクチャに合わせて」といった制約を加えたりします。
  • 制約の指定: 「TypeScriptで記述し、ユニットテストも追加してください」「パフォーマンスを考慮した実装にしてください」など、要件を詳細に指定します。
  • 段階的な指示: 複雑なタスクは一度に全てを任せるのではなく、小さなステップに分割して指示することで、より正確な結果を得られます。

2. コード補完・リファクタリング AntigravityのAIエージェントは、リアルタイムでコードを解析し、高度な補完やリファクタリング提案を行います。

  • インライン補完: 入力中にAIが予測したコードスニペットが自動で表示されます。Tabキーで受け入れ、Escキーで拒否します。
  • ブロックコード生成: 特定の関数やクラスの定義、複雑なロジックをChatパネルで指示すると、AIがコードブロック全体を生成します。
  • リファクタリングの提案: 複雑な関数を分割する、可読性を高める、パフォーマンスを改善するといったリファクタリングを自動で提案します。エディタの電球アイコンや右クリックメニューから適用できます。

3. テストコード生成と実行 高品質なコードにはテストが不可欠です。Antigravityはテストコードの生成と実行も強力にサポートします。

  • テストコードの自動生成: Chatパネルで「このUserServiceクラスに対してユニットテストを生成してください」と指示するだけで、AIが適切なテストフレームワーク（Jest, Pytestなど）に応じたテストコードを生成します。
  • テストの実行とフィードバック: 生成されたテストをAntigravity内で直接実行し、結果をChatパネルで確認できます。失敗したテストについては、AIが原因の特定と修正案を提案します。
  • テスト駆動開発（TDD）の支援: まずテストケースを生成し、そのテストがパスするようにAIにコードを書かせる、といったTDDワークフローもスムーズに実行できます。

4. バージョン管理システム（Git）との連携 AntigravityはGitとの統合が非常に強力です。

  • 自動コミット提案: AIエージェントがコードの変更を検知し、適切なコミットメッセージを自動で生成し、コミットを提案します。
  • プルリクエスト作成支援: 新しい機能やバグ修正が完了すると、AIが変更内容に基づいたプルリクエストのタイトル、説明、レビュアーの提案まで行います。
  • 競合解決支援: マージコンフリクトが発生した場合、AIが競合箇所を特定し、解決策を提案します。

他のAI IDEとの比較

現在、多くのAI IDEやAIアシスタントが存在しますが、Antigravityは特にAIエージェントの自律性とマルチLLM対応において独自の強みを持っています。主要な競合製品と比較してみましょう。

機能/製品名 Antigravity Cursor Windsurf (Google) GitHub Copilot Claude Code (Anthropic)
AIエージェントの自律性 高（タスク全体を計画・実行） 中（コード生成・編集に特化） 中（特定のタスクを支援、高度なAIチャット） 低（コード補完が主） 中（長文のコード生成・理解に強い）
マルチLLM対応 対応（Gemini, Claude, GPT等を選択可能） 一部対応（GPT-4、Claudeを統合） Google製のLLMに特化 OpenAIのLLMに特化 Anthropic Claudeに特化

Manager View あり（エージェントの思考プロセス可視化） なし なし（ただし、高度なAIインタラクション機能） なし なし

価格モデル フリーミアム（基本機能無料、高度なAI機能は有料） フリーミアム（基本機能無料、高度なAI機能は有料） サブスクリプション サブスクリプション API利用料、またはサブスクリプション

Antigravityの最大の差別化要因は、AIエージェントが開発タスクを「自律的に」処理し、その思考プロセスを「Manager View」で可視化できる点にあります。これにより、開発者はAIの行動を理解し、的確に介入・指導できるようになります。また、マルチLLM対応により、特定のLLMに依存することなく、常に最適なAI能力を開発に投入できる柔軟性も大きな強みです。

エージェント機能活用テクニック

Antigravityの真価は、その強力なAIエージェント機能にあります。ここでは、その実践的な活用テクニックを紹介します。

自律コーディング：開発タスクを任せる

Antigravityのエージェントは、単なるコード生成ツールではありません。複雑な開発タスク全体を任せ、自律的に完了させることができます。

実践シナリオ例：新しい機能の追加 1. タスクの指示: Chatパネルで「ユーザー認証機能に二段階認証（MFA）を追加してください。既存の認証モジュールauth.pyを拡張し、totpライブラリを使用してください。」と具体的に指示します。 2. エージェントの計画立案: エージェントは、Manager Viewで以下のような計画を提示します。 * auth.pyの修正案の生成（MFAロジックの追加） * ユーザーモデルへのMFA設定フィールドの追加（DBスキーマ変更） * MFA有効化/無効化APIエンドポイントの作成 * 関連するユニットテストの生成と実行 * ドキュメントの更新 3. 計画の承認と実行: 開発者はManager Viewで計画を確認し、必要であれば修正指示を与えます。承認後、エージェントは計画に従って自律的にコードを生成し、テストを実行します。 4. 進捗監視と介入: エージェントはリアルタイムで進捗を報告し、問題が発生した場合は開発者に介入を求めます。例えば、「totpライブラリが見つかりません。pip install pyotpを実行しますか？」といった確認です。 5. レビューとデプロイ提案: タスク完了後、エージェントはコードの変更点、テスト結果、更新されたドキュメントをまとめ、プルリクエストの作成を提案します。

Manager View：エージェントの思考プロセスを可視化

Manager Viewは、AIエージェントが何を考えて、どのように行動しているかを「見える化」するAntigravity独自の機能です。これを活用することで、AIをより効果的に指導し、協調開発の質を高めることができます。

  • エージェントの思考ステップ: Manager Viewでは、エージェントがタスクを解決するために立てた「計画」、各ステップでの「思考（Thought）」、そして実行された「アクション（Action）」が時系列で表示されます。これにより、AIがなぜそのコードを書いたのか、なぜそのライブラリを選んだのかといった背景を理解できます。
  • リアルタイムフィードバック: エージェントの思考プロセス中に、「このアプローチは変更してください」「もっと効率的なアルゴリズムがあります」といった具体的なフィードバックをManager Viewから直接入力できます。エージェントはそれを受けて計画を修正し、再実行します。
  • デバッグと問題解決: エージェントがバグに直面した場合、Manager Viewでエラーログやデバッグセッションの進行状況を確認できます。AIが問題を誤って解釈している場合は、正しい方向へ誘導することで、解決までの時間を短縮できます。
  • 学習とカスタマイズ: 特定のプロジェクトやチームのコーディング規約に沿わない行動をした場合、Manager Viewから学習させることができます。「このパターンではなく、HogeUtilクラスを使ってください」といった具体的な指示により、エージェントの行動をカスタマイズし、将来的にはより最適な提案をさせることが可能になります。

ブラウザ統合テスト：エンドツーエンドの検証

Antigravityは、Webアプリケーション開発において特に強力なブラウザ統合テスト機能を提供します。これは、実際のブラウザ環境でアプリケーションのUIとロジックをエンドツーエンドでテストする機能です。

  • テストシナリオの自動生成: Chatパネルで「ユーザー登録ページのフォーム入力と送信、成功メッセージの表示を確認するE2Eテストを生成してください」と指示します。エージェントはSeleniumやPlaywrightなどのフレームワークを用いて、テストコードを生成します。
  • 仮想ブラウザでの実行: Antigravityのクラウド環境には仮想ブラウザが統合されており、生成されたE2Eテストを直接実行できます。テストの実行状況やUIのスクリーンショット、動画記録などもManager Viewで確認可能です。
  • UI変更への自動対応: UI要素のセレクタ変更など、Web UIがアップデートされた場合でも、エージェントはテストコードを自動的に解析・修正し、テストの保守コストを削減します。
  • CI/CDパイプラインとの連携: Antigravityのブラウザ統合テストは、Google Cloud BuildなどのCI/CDパイプラインに容易に組み込むことができ、デプロイ前の品質保証を自動化します。

よくある質問（FAQ）

Q1: Antigravityは無料ですか？

A1: Antigravityは「フリーミアム」モデルを採用しています。基本的なコードエディタ機能やAIアシストの一部は無料で利用できます。しかし、高度なAIエージェントによる自律コーディング、マルチLLMの利用、Manager Viewによる詳細な思考プロセス可視化、大規模なクラウドリソースの利用など、プロフェッショナルな開発者向けの機能は有料のサブスクリプションプランで提供されます。個人開発者向けのStartupプランから、大規模チーム向けのEnterpriseプランまで、様々なニーズに対応したプランが用意されています。

Q2: Antigravityはどのような開発言語に対応していますか？

A2: Antigravityは、2026年現在、主要な開発言語のほぼ全てに対応しています。特にJavaScript/TypeScript (React, Angular, Vue.js), Python (Django, Flask, FastAPI), Go, Java (Spring Boot), C#, PHPといったWebアプリケーション開発やバックエンド開発でよく使われる言語・フレームワークに強いAIサポートを提供します。また、モバイル開発（Kotlin/Swift）やデータサイエンス（Jupyterノートブック統合）などもサポートされており、今後のアップデートで対応言語はさらに拡大する予定です。

Q3: 既存のプロジェクトをAntigravityに移行するにはどうすれば良いですか？

A3: 既存のプロジェクトをAntigravityに移行する方法はいくつかあります。最も一般的なのは、Gitリポジトリからのインポートです。GitHub、GitLab、BitbucketなどのリポジトリURLを指定し、認証を行うだけで、Antigravityは自動的にプロジェクトをクローンし、開発環境を構築します。VS Code拡張版をご利用の場合は、既存のローカルワークスペースを直接開くことも可能です。また、Google DriveやGoogle Cloud Storageとの連携により、クラウドストレージに保存されたプロジェクトを直接開いて作業することもできます。
